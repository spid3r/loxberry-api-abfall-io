Abfallabholung (api.abfall.io)

Die Einstellungen passen Sie in der LoxBerry-Verwaltung an (Standort, Abruf, optional MQTT und Loxone-Links). Diese Seite liefert dieselben Abholtermine per HTTP.

Typische Wege (im Plugin genauer erklärt):

  • Loxone — loxone.php-Links im virtuellen HTTP-Eingang: Ihr Miniserver holt in einem festen Rhythmus eine Zahl (Tage bis zur Leerung) pro Tonne, ideal, wenn alles in Loxone laufen soll.
  • MQTT — nach jedem Plugin-Abruf werden die Werte an einen Broker gesendet, mehrere Ziele (App, Heim-Automation) können zugreifen, ohne loxone.php einzurichten.
  • JSON — dieselben Daten in einem Rutsch: an die Seitenadresse ?format=json anhängen (nach dem ersten erfolgreichen Abruf im Plugin).

Hinweis: Dieses Plugin ist eine unabhängige Community-Lösung, nicht offiziell von den Betreibern von api.abfall.io. Es gibt keinen Anspruch auf Support durch die API-Betreiber oder die Autoren. Automatische Abrufe sind auf mindestens 6 Stunden Abstand begrenzt. Der Dienst kann jederzeit ausfallen oder sich ändern — Nutzung auf eigenes Risiko. Details: DISCLAIMER.md im GitHub-Repository des Plugins.
