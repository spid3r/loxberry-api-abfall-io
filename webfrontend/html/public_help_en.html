Waste collection (api.abfall.io)

Configure the plugin in LoxBerry admin (location, fetch, optional MQTT and Loxone links). This public URL serves the same collection data over HTTP.

Typical use (the plugin has more detail in Settings):

  • Loxone — use the loxone.php links in a Virtual HTTP Input: the Miniserver polls and gets a number of days per waste type; best for pure Loxone workflows.
  • MQTT — after each fetch, values are published to topics so several systems can subscribe without each setting up loxone.php.
  • JSON — add ?format=json to this page for a full snapshot (after at least one successful fetch in the plugin).
